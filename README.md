# automacao_de_graficos

Uma automação que consiste em receber dados de uma empresa de CallCenter, onde que o arquivos está com dados de um periodo de um ano de uma empresa, esses dados será representados como gráfico e poderão ser filtrandos, e podendo ser mostrado no que pode ser feito para reduzir os cancelamentos.

Bibliotecas usadas: pandas, plotly
